Complete Documentation of My Learning Journey

Day 1 - Documentation

date: 25/03/2025

On the first day of this module, I learned about basics of HTML (HyperText Markup Language). Starting with creating my first file named as Index.html in VS CODE. After creating a file I used multiple tags including:

  • Heading tag (from h1 to h6)
  • Paragraph tag (to write content)
  • Anchor tag (adds a link)
  • Image tag (to add an image)
  • BR tag (breaks the line)

The interactive and hands-on learning was very effective for me .

Day 2 - Documentation

date: 26/03/2025

On the second day, I was tutored about more details about HTML tags such as:

  • Bold, italic & underline tag(edits font)
  • Big & small tag(changes font size)
  • Section tag(divides webpage into sections)
  • HR tag(creates a horizontal line on the page)
  • Subscript & superscript
  • Pre tag
  • Article tag(to write article)
  • Aside tag(to aside from main content)
  • Div tag(creates a block)
  • Span tag(creates a block for specific space)

I also learned about page layout techniques and the use of header, main and footer.

Day 3 - Documentation

date: 27/03/2025

On Day 3 I learned about advanced tags like lists and tables in HTML.
Where list is of two types: Ordered list and Unordered list.
Tables in HTML, includes tags like:

  1. tr - displays table row
  2. td - displays table data
  3. th - displays table header

I also picked up on table caption,thead and tbody tags, forms and labels.After this we concluded our first our first topic of the module i.e HTML .

Day 4 - Documentation

date: 28/03/2025

Day 4 started with a new topic: **CSS (Cascading Style Sheets)**. We learned how to link a CSS file with an HTML file using the link tag. I also learned how to style text using various CSS attributes like background color, font, size of text, etc.

Day 5 - Documentation

date: 29/03/2025

Day 5 focused on how to create a box and enclose text in it and style it differently using CSS. After this, we concluded CSS and moved to JavaScript, where we had an introductory session on JavaScript, explaining its uses and purpose.

Day 6 - Documentation

date: 30/03/2025

On Day 6, I learned the basics of **JavaScript**, including its syntax, data types, variables, and basic operations. I also learned about how JavaScript is used to make web pages interactive and dynamic. This was an introduction to how JavaScript can manipulate HTML and CSS elements dynamically.
